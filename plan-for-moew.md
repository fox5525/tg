# ระบบบันทึกข้อมูลแพนกวิน (Penguin Data Recording System)

## 🛠 Tech Stack
**Backend:**
- Node.js (Express.js)
- RESTful API
- Database: PostgreSQL (หรือ MySQL)

**Frontend:**
- Vue.js + Vite
- TailwindCSS + DaisyUI

**DevOps & Collaboration:**
- Docker + Docker Compose (ทำ Dockerfile สำหรับทั้ง Backend และ Frontend)
- GitHub Collaboration (แบ่ง Branch ทำงาน สำหรับทีม 2 คน: Frontend 1 คน, Backend 1 คน)

---

## 📌 ฟีเจอร์ของระบบ (System Features)
- ระบบจัดการข้อมูล (CRUD): เพิ่ม, ลบ, แก้ไข, ค้นหา ข้อมูลแพนกวิน
- ระบบฟิลเตอร์ (Filter): สามารถกรองข้อมูลตาม เพศ / สี / อายุ / สายพันธุ์ / ถิ่นที่อยู่อาศัย / สถานะ
- การแสดงผล (Display): สามารถเลือกการแสดงผลข้อมูลเป็นแบบ การ์ด (Card) หรือ ตาราง (Table) ได้

---

## 🐧 โครงสร้างข้อมูลแพนกวิน (Penguin Data Fields)
- **เพศ (Gender):** ตัวผู้, ตัวเมีย, ไม่ทราบ
- **สี (Color):** ดำ-ขาว, มีแถบเหลือง/ส้ม, สีน้ำเงินเทา, อื่นๆ (เลือกได้มากกว่า 1 สี)
- **ช่วงอายุ (Age Group):** ลูกนก (Chick), วัยรุ่น (Juvenile), ตัวเต็มวัย (Adult) (เลือกได้มากกว่า 1 ช่วงอายุ)
- **สายพันธุ์ (Species):** แพนกวินจักรพรรดิ (Emperor), แพนกวินคิง (King), แพนกวินอาเดลี (Adélie), แพนกวินชินสแตรป (Chinstrap), แพนกวินเจนทู (Gentoo), แพนกวินแมกเจลแลนิก (Magellanic), แพนกวินแอฟริกัน (African), แพนกวินนางฟ้า (Fairy/Little Blue), และอื่นๆ
- **ถิ่นที่อยู่อาศัย / โซนจัดแสดง (Habitat / Zone):** ระบุข้อความ (ไม่บังคับ)
- **สถานะ (Status):** แข็งแรงปกติ, ป่วย/กำลังรักษา, อยู่ระหว่างอนุบาล, ส่งตัวไปที่อื่น, เสียชีวิตแล้ว, ปล่อยคืนสู่ธรรมชาติ (เลือกได้มากกว่า 1 สถานะ)
- **รูปภาพ (Image):** อัปโหลดรูปภาพแพนกวิน (ไม่บังคับ)

## 👥 ระบบสิทธิ์การใช้งาน (Roles & Permissions)

### 👑 ฟีเจอร์ Admin (ผู้ดูแลระบบ)
- จัดการผู้ใช้งานระบบ (เพิ่ม/ลบ/แก้ไข/กำหนดสิทธิ์)
- จัดการข้อมูลแพนกวินทั้งหมดในระบบ
- จัดการข้อมูลสายพันธุ์และโซนจัดแสดง
- ดูรายงานและสถิติภาพรวมของแพนกวินในระบบ

### 👨‍⚕️ ฟีเจอร์ Staff (เจ้าหน้าที่ผู้ดูแล / Zookeeper)
- จัดการข้อมูลแพนกวิน (เพิ่ม/ลบ/แก้ไข)
- อัปเดตสถานะสุขภาพและความเป็นอยู่ของแพนกวิน
- อัปเดตข้อมูลสถานที่และโซนจัดแสดงของแพนกวิน

### 👤 ฟีเจอร์ User (ผู้ใช้งานทั่วไป / ผู้เยี่ยมชม)
- สมัครสมาชิก (Register)
- ล็อกอินเข้าสู่ระบบ (Login)
- ดูข้อมูลและรายละเอียดของแพนกวิน (เฉพาะที่เปิดเผยได้)
- ค้นหาและดูข้อมูลสายพันธุ์ของแพนกวิน
- ดูรูปภาพแพนกวิน
